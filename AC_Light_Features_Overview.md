# AC Light — обзор возможностей

Лёгкое офлайн-приложение для мастеров по обслуживанию автомобильных кондиционеров.

Главный принцип: **Открыл → Нашёл клиента → Добавил запись → Закрыл.** Никаких лишних экранов, никакой обязательной регистрации, никакого интернета — вся база клиентов хранится локально на устройстве мастера.

---

## 📱 О проекте

AC Light — это не универсальная CRM, а узкоспециализированный инструмент для одной задачи: быстро найти клиента, посмотреть историю обслуживания его автомобиля и добавить новую запись о визите за 15–20 секунд.

Приложение построено по принципу трёхуровневой линейки продукта:

- **AC Light (Free)** — полнофункциональный офлайн-CRM: клиенты, авто, история визитов, реферальная система (включая QR-коды и сканирование номерных знаков), резервное копирование.
- **AC Light Pro** — платные расширения для тех, кто хочет вести бизнес осознанно: автоматический расчёт стоимости визита по прайс-листу, скидки, статистика доходов.
- **AC Cloud** *(в перспективе)* — облачная синхронизация между устройствами, мультимастера, Telegram-бот для реферальной программы.

## ✨ Основные возможности

### Клиенты и автомобили
- Быстрый живой поиск по имени, телефону, марке, модели, госномеру, VIN, двигателю, заметкам
- Карточка клиента с историей всех визитов
- До 8 фото автомобиля на карточку, с каруселью и просмотром с зумом
- Импорт контакта из телефонной книги одним тапом

### Реферальная система
- Каждый клиент может привести другого — связь фиксируется автоматически
- Генерация и шаринг QR-кода клиента
- Сканирование QR-кода при создании нового клиента — реферер подставляется мгновенно
- Счётчик "сколько клиентов привёл" прямо в карточке

### Распознавание номерного знака
- Скан номера камерой — если клиент уже есть в базе, сразу открывается форма нового визита с уже прикреплённым фото автомобиля
- Если клиента нет — предлагается создать карточку с уже распознанным номером
- Номер также автоматически распознаётся из обычных фото авто, если поле ещё не заполнено

### Резервное копирование
- Экспорт всей базы (данные + фото) в один ZIP-архив через системное меню "Поделиться"
- Импорт/восстановление из ранее созданной копии
- Никакой автоматической отправки в облако — вы сами решаете, куда сохранить копию

### Безопасность и удобство
- Опциональная блокировка приложения по отпечатку пальца / Face ID / PIN
- Мультиязычный интерфейс: русский, украинский, английский, немецкий, польский, испанский, французский, итальянский, турецкий
- Вибро- и звуковая обратная связь на действия
- Полностью тёмная тема

### AC Light Pro
- Настраиваемый прайс-лист (хладагент, масло, кольца, краситель, диагностика)
- Автоматический расчёт стоимости визита со скидками
- Статистика: выручка и количество визитов за день/месяц/год, график по месяцам, топ рефереров
