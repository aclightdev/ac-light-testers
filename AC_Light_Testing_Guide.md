# AC Light — инструкция по функционалу для тестировщиков

Это руководство поможет разобраться, что умеет приложение, и что стоит проверить в первую очередь.

---

## Главный экран (поиск)

При открытии приложения вы сразу попадаете на экран поиска.

- **Строка поиска** — начните вводить имя, телефон, марку авто или госномер, результаты появятся сразу, без нажатия каких-либо кнопок.
- **Кнопка "+"** вверху — создание нового клиента.
- **Иконка камеры** рядом с поиском — сканирование номерного знака (наведите камеру на номер автомобиля, если он уже есть в базе — откроется форма нового визита; если нет — предложит создать нового клиента).
- **Иконка статистики** — доступна только в платной версии Pro (в бесплатной версии покажет, что это Pro-функция).
- **Шестерёнка** — настройки.

**Что проверить:** попробуйте найти клиента по разным критериям — имени, куску телефона, марке машины.

---

## Создание клиента

Нажмите "+". Форма попросит:

- **Фото автомобиля** (сверху) — можно сфотографировать через камеру или выбрать из галереи, до 8 фото.
- **Госномер** и **марка** — единственные по-настоящему обязательные поля.
- **Имя** — тоже обязательно.
- Остальное (фамилия, телефон, модель, двигатель, год, госномер, VIN) — по желанию.
- **"Кто пригласил"** — можно найти существующего клиента через поиск, отсканировать его QR-код, или просто вписать имя вручную, если человека нет в базе.
- **Импорт из контактов** — кнопка рядом с именем/телефоном, подтягивает данные из телефонной книги одним тапом.

**Что проверить:** попробуйте создать клиента с минимумом полей (только имя + марка), и отдельно — с полным набором данных.

## Карточка клиента

Откройте любого клиента из списка поиска.

- Сверху — данные клиента и авто, с каруселью фото (если их несколько — можно листать, тапнуть для увеличения и зума).
- Кнопка со стрелкой — телефон кликабельный, сразу открывает набор номера.
- Карандаш — редактирование клиента.
- Корзина — удаление (спросит подтверждение).
- Иконка QR — показать свой QR-код клиента для передачи другим (по нему можно "привести" нового клиента).
- Ниже — история визитов, с кнопкой "+ Добавить визит".

**Что проверить:** отредактируйте клиента, попробуйте удалить (можно создать тестового специально для этого), проверьте, что кнопка звонка работает.

---

## Визиты

Кнопка "+ Добавить визит" в карточке клиента открывает форму:

- Дата (по умолчанию сегодня)
- Хладагент, масло — количество
- Переключатели услуг (UV-краситель, замена колец, диагностика)
- Оплата — вводится вручную (в платной Pro-версии считается автоматически по прайс-листу)
- Заметки

**Что проверить:** попробуйте сохранить полностью пустой визит — должно показать ошибку и не дать сохранить. Отредактируйте существующий визит, попробуйте удалить.

## Реферальная система

- В карточке клиента — кнопка "Показать QR", можно поделиться картинкой через любой мессенджер.
- При создании нового клиента — кнопка сканирования QR в поле "Кто пригласил", наведите камеру на чужой QR-код, реферер подставится автоматически.

**Что проверить:** создайте двух клиентов, у одного сгенерируйте QR, у второго при создании отсканируйте этот QR — должна появиться связь "кто пригласил".

## Резервное копирование (Настройки → Резервная копия)

- **"Создать резервную копию"** — соберёт все данные в один файл и предложит отправить его куда угодно (в свой Google Диск, Telegram "Избранное" и т.п.).
- **"Восстановить из копии"** — выбираете ранее созданный файл, все данные вернутся к тому состоянию.

**Что проверить:** создайте копию, отправьте себе в Telegram, убедитесь что файл дошёл.

## Настройки

- **Язык** — 9 языков на выбор, переключается сразу, без перезапуска.
- **Вибрация / Звук** — обратная связь на действия, можно включить/выключить.
- **Безопасность** — блокировка приложения по отпечатку/PIN (по желанию).
- **Privacy Policy** — политика конфиденциальности, доступна даже без интернета.

**Что проверить:** переключите язык на любой не самый очевидный (например Türkçe или Polski) — весь интерфейс должен перевестись.
